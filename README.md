# De que trata.
Esta actividad fue realizada en el salon de clases con los códigos que el profesor nos asigno, el objetivo de este proyecto es realizar una "agenda médica" en la cual se puedan checar que citas estan agendadas, los medicos asignados y el estado de la cita.

# Que se hizo

Se creo en el visual Studio community un nuevo proyecto utilizando la arquitectura MVC, dentro del proyecto se crearon las clases medico, citas, paciente, los controladores de las mismas clases, del mismo modo se agraron las vistas de cada una de las clases, cada clase permite registrar los valores de cada uno de las entidades, del mismo modo se creo una carpeta Data en la cual se crearon archivos .json los cuales registran los datos en memoria, tambien se le agrego una barra de navegacion lo cual permite encontar las opciones de la manaera más cómoda posible.

# Como funciona.
El objetivo de este proyecto es registrar citas médicas, del mismo modo poder asognar los medicos, el motivo de la cita y el estado de la misma por ejemplo; si la cita esta confirmada, pendiente o cancelada.
